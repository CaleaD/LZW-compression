# LZW-compression
Implementation of a compression algorithm, LZW (Lempel-Ziv-Welch Algorithm).

The application supports both compression and decompression. 

| Command  | Definition |
| ------------- | ------------- |
| -c [FileName] | Compress File  |
| -d  | Decompress File  |
| -e  | Check encoding  |
| -h  | User commands (help)  |
| -q  | Exit the application  |
